# 管理后台数据可视化图表预览

## 已实现的图表

### 1. 用户增长趋势图（折线图）
- 展示10个月的用户增长数据
- 使用Recharts LineChart组件
- 清晰展示用户数量随时间的增长趋势

### 2. 贷款申请状态分布（饼图）
- 展示已批准、审批中、已拒绝三种状态的分布
- 使用Recharts PieChart组件
- 带百分比标签，不同颜色区分状态

### 3. 贷款产品分布（柱状图）
- 展示短途游贷、毕业旅行贷、研学游贷、跨省游贷的申请数量
- 使用Recharts BarChart组件
- 直观对比不同产品的受欢迎程度

### 4. 还款情况分析（堆叠柱状图）
- 展示6个月的按时还款和逾期还款数据
- 使用Recharts BarChart组件的堆叠模式
- 绿色表示按时还款，红色表示逾期还款

## 技术实现

- 使用Recharts库（React图表库）
- 响应式设计，自适应不同屏幕尺寸
- 统一的配色方案，符合整体UI风格
- 图表数据可通过后端API动态更新

## 数据来源

当前使用模拟数据展示，实际使用时需要：
1. 后端提供统计API接口
2. 前端调用API获取真实数据
3. 实时更新图表展示
